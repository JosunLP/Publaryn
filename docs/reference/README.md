# Reference Hub

Use this section when you need authoritative documentation for the currently
supported Publaryn 1.0 surface.

## Primary references

- [1.0 release contract](/1.0)
- [API and adapter route reference](/api-routes)
- [ADR index](/adr/README)
- [Release checklist](/release-checklist)

## What belongs here

- control-plane route groups and native adapter mounts
- release support and compatibility expectations
- architecture decisions with ongoing impact on public behavior
- operational runbooks and release-note artifacts

## Recommended workflow for contributors

1. Check the [1.0 release contract](/1.0) before changing public behavior.
2. Read the [ADR index](/adr/README) before touching auth, visibility, governance, or protocol logic.
3. Update the [API route reference](/api-routes) when adding or changing mounted surfaces.
4. Review the [release checklist](/release-checklist) before tagging a stable release.
